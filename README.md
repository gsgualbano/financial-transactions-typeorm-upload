### ğŸ  [Homepage](https://github.com/gsgualbano/financial-transactions-typeorm-upload#readme)

## Install

```sh
yarn
```

## Usage

first run the migrations to create the tables on the database(postgres)
```sh
yarn typeorm migration:run
```

then
```sh
yarn dev:server
```
to start the server

## Run tests

```sh
yarn test
```
